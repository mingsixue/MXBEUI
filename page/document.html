IconFont 图标

font-class引用

font-class是unicode使用方式的一种变种，主要是解决unicode书写不直观，语意不明确的问题。

与unicode使用方式相比，具有如下特点：

  1. 兼容性良好，支持ie8+，及所有现代浏览器。
  2. 相比于unicode语意明确，书写更直观。可以很容易分辨这个icon是什么。
  3. 因为使用class来定义图标，所以当要替换图标时，只需要修改class里面的unicode引用。
  4. 不过因为本质上还是使用的字体，所以多色图标还是不支持的。

使用步骤如下：

第一步：引入项目下面生成的fontclass代码：

<link rel="stylesheet" type="text/css" href="../fonts/iconfont.css">

第二步：挑选相应图标并获取类名，应用于页面：

<i class="iconfont icon-xxx"></i>
"iconfont"是你项目下的font-family。可以通过编辑项目查看，默认是"iconfont"。
富文本编辑器

WangEditor

集成了WangEditor编辑器，轻量级 web 富文本编辑器，配置方便，使用简单。支持 IE10+ 浏览器。

官网地址：http://www.wangeditor.com/

文档：www.kancloud.cn/wangfupeng/wangeditor3/332599

使用说明：

使用步骤如下：

第一步：引入 wangEditor样式代码（可按需修改）：

<link rel="stylesheet" type="text/css" href="../js/libs/wangEditor.css">

第二步：放置编辑器HTML代码：

<div class="editor-wrap">
    <div id="editor" name="description"></div>
</div>

第三步：引入文件实例化编辑器:

<script src="../js/libs/wangEditor.min.js"></script>
//初始化编辑器
var E = window.wangEditor;
var editor = new E('#editor');
    editor.customConfig.menus = [
        'head',
        'bold',
        'italic',
        'underline',
        'strikeThrough',
        'foreColor',
        'backColor',
        'link',
        'list',
        'justify',
        'quote',
        'table',
        'image',
        'undo',
        'redo'
    ];
    editor.create();

具体编辑器相关用法查看 wangEditor 官网。

可自行集成其他编辑器，去掉相应css、js文件即可
编辑器的 z-index 设为了10000，弹窗需要设为比编辑器 z-index 层级高。
日期控件

My97DatePicker

集成了功能强大的日期控件 My97DatePicker。

官网地址：http://www.my97.net/

文档：http://www.my97.net/demo/index.htm

使用说明：

使用步骤如下：

第一步：引入 My97DatePicker库：

<script src="../js/libs/My97DatePicker/WdatePicker.js"></>

第二步：HTML绑定日期函数：

<input type="text" onClick="WdatePicker()" />

具体相关用法查看 My97DatePicker 官网。

可集成其他日期插件，去掉相应css、js文件即可
